# Estructura de Directorio

La estructura del proyecto es la siguiente, mostrada de manera resumida:

.
src
├── modulos
│   ├── base-datos
│   ├── formulario-contacto
│   ├── recaptcha
│   └── servicios
│   │   ├── bitacora
│   │   ├── link-navegacion
│   │   ├── multimedia
│   │   └── pagina
├── public
├── rutas
│   └── index.js
└── index.js
  • modulos/base-datos: Se encuentra la conexión a la base de datos.
  • modulos/formualario-contacto: Encargado de la función del formualario de contacto.
  • modulos/recaptcha: Encargado del funcionamiento de recatcha de google.
  • modulos/servicios/bitacora: Encargado de las funciones de la bitácora.
  • modulos/servicios/link-navegacion: Encargado de las funciones de los link externos.
  • modulos/servicios/pagina: Encargado de las funciones del contenido de la página.
  • public: Directorio donde se almacenan las imagenes.
  • rutas/index.js: Archivo que administra las rutas de la API.
  • index.js: Archivo principal donde se levanta el servidor.
