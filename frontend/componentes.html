# Componentes

# AppGridConjunto

Actualiza la data de secciones tipo:

  • Articulo
  • Slide
  • Testimonio

Utiliza los componentes:

  • AppActualizarEstructuraTexto
  • AppActualizarContenido
  • AppAgregarContenido

# AppGridContenido

Actualiza la data de secciones tipo:

  • Descripcion
  • Hero

Utiliza el componente:

  • AppActualizarContenido

# AppGridSedes

Actualiza la data de la sección Sedes.

Utiliza el componente:

  • AppActualizarSede
  • AppActualizarMultimedia
  • AppAgregarMultimedia

# AppGridBitacora

Muestra la data de la bitacora

# AppActualizarColegio

Actualiza la data básica del colegio:

  • Nombre
  • Correo
  • Teléfono

# AppActualizarConjunto

  • Actualiza secciones tipo Conjunto
  • Componente hijo: AppActualizarConjuntoArticulo

# AppActualizarConjuntoArticulo

Actualiza en el Front el artículo seleccionado.

# AppActualizarContenido

  • Actualiza secciones tipo Contenido
  • Métodos:
    • URI: api/servicio/subir-multimedia, Tipo: POST
    • URI: api/servicio/actualizar/contenido, Tipo: POST

# AppActualizarEstructuraTexto

  • Actualiza estructuras de texto.
  • Método:
    • URI: api/servicio/actualizar/estructura-texto, Tipo: POST

# AppActualizarMultimedia

  • Actualiza multimedia.
  • Método:
    • URI: api/servicio/actualizar/multimedia, Tipo: POST

# AppActualizarNavegacion

  • Actualiza los links externos.
  • Método:
    • URI: api/servicio/actualizar/navegacion, Tipo: POST

# AppActualizarSede

  • Actualiza las sedes.
  • Método:
    • URI: api/servicio/actualizar/sede, Tipo: POST

# AppAgregarContenido

  • Agrega contenido.
  • Métodos:
    • URI: api/servicio/subir-multimedia, Tipo: POST
    • URI: api/servicio/agregar/contenido, Tipo: POST

# AppAgregarMultimedia

  • Agrega multimedia.
  • Método:
    • URI: api/servicio/agrega/multimedia, Tipo: POST
