Nota

Como se nombren los archivos serán tomados en el router de la página web.

# Index (Inicio)

Página de inicio o home, está pagina está encargada de mostrar el inico de la web.

  • Ubicación: ./pages/index.vue

# Data

Los datos se cargan en vuex.

  • Obtención:
    computed: {
  ...mapGetters({
    slide: "contenido/slide",
    articulo: "contenido/articulo",
    testimonio: "contenido/testimonio",
    galeria: "contenido/galeria",
  }),
},
async fetch({ store, params }) {
  await store.dispatch("contenido/getConjunto", "index");
  await store.dispatch("contenido/getGaleria", "index");
},
  • Ubicación: ./store/contenido.js
  • Acction:
    • getContenido("index")
    • getGaleria("index")

# Data

Los datos se cargan en vuex.

  • Obtención:
    computed: {
  ...mapGetters({
    sedes: "contenido/sedes",
  }),

  sede() {
    let data = this.sedes.filter((el, index) => {
      return index == this.selection;
    });
    return data[0];
  },
},

async fetch({ store, params }) {
  await store.dispatch("contenido/getSedes");
},
