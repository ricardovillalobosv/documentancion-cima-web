# Páginas para Administración

# Login

Encargada de logear a un usuario al Dashboard. Para hacer el logueo se utilizó el siguiente método:

  • Ubicación: ./pages/login.vue

  • Método:

    • Tipo: POST
    • URL: http://intranet.cima.com.pe:3000/api/cimapersonal/personal/login

# Clave

Encargada de validar la clave privada del Dashoboard.

  • Ubicación: ./pages/clave.vue

# Dashboard

Archivo principal donde se muestra las secciones para administrar la página web.

  • Ubicación: ./pages/dasboard/index.vue

# Dashboard - Articulo

Subpágina de Dasboard donde se administrarán los artículos de la página web.

Hace uso del componente AppGridConjunto

  • Ubicación: ./pages/dasboard/articulo.vue

# Data

Los datos se cargan en vuex.

  • Obtención:
    computed: {
  ...mapGetters({
    articulo: "contenido/articulo",
    paginaSeleccionada: "mantenimiento/paginaSeleccionada"
  })
},

methods: {
  async Contenido() {
    await this.$store.dispatch(
      "contenido/getConjunto",
      this.paginaSeleccionada
    );
  }
},
  • Ubicación: ./store/contenido.js
  • Acction: getConjunto("...")

# Dashboard - Galeria

  • Ubicación: ./pages/dasboard/galeria.vue

Hace uso de los componentes:

  • AppActualizarEstructuraTexto
  • AppActualizarMultimedia
  • AppAgregarMultimedia

# Data

Los datos se cargan en vuex.

  • Obtención:
    computed: {
  ...mapGetters({
    galeria: "contenido/galeria",
    paginaSeleccionada: "mantenimiento/paginaSeleccionada",
  }),

  getGaleria() {
    if (this.galeria.length > 0) return this.galeria[0];
    else return null;
  },
},

methods: {
  async Contenido() {
    await this.$store.dispatch(
      "contenido/getGaleria",
      this.paginaSeleccionada
    );
  },
  //...
},
  • Ubicación: ./store/contenido.js
  • Acction: getContenido("...")
